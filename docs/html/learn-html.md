## 开始
- 超文本标记语言

- 基本标签
```<!DOCTYPE html> ```：H5语法   
```<p></p>```：段落标记   
```<h2></h2>```：标题字   
```<br>```：换行（独目标记）   
```<hr color="red" width="50%">```：分割线   
```<pre></pre>```：预留格式   
```<del></del>```：删除字   
```<ins></ins>```：插入字   
```<b></b>```：粗体字   
```<i></i>```：斜体字   
```<font color="red" size="5"></font>```：字体标签
